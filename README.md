# Canteen Food Demand Prediction using Machine Learning

##  Project Overview
This project predicts daily food demand in a college canteen using Machine Learning.
It helps reduce food wastage and improve planning.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
Sample dataset created based on:
- Day of week
- Weather
- Exam days
- Student footfall

##  ML Model
- Linear Regression
- Random Forest (optional)

## How to Run
1. Clone the repository
2. Install required libraries
3. Run `main.py`
